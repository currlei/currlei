### 🏛️ CitizenCentral: Integrated Government Service Request System

> **Roles:** Project Lead &middot; Systems Analyst &middot; Technical Writer
> **Academic Submission:** BSIS Project (AY 2025–2026)

An enterprise-grade e-governance platform designed to eliminate manual data silos and bureaucratic bottlenecks across Philippine Local Government Units (LGUs). As the Project Lead, I architected the core system specifications, engineered the Role-Based Access Control (RBAC) security framework, and drafted the compliance guidelines for national data privacy standards.

<p align="center">
  <img src="https://github.com/user-attachments/assets/1349de91-a095-4407-a818-872094ea95ed" width="350" alt="Diagram SADD (1)" />
  <br><br>
  <a href="https://docs.google.com/document/d/1M8Omc68aAyVYBRHjzzJYz8xYhUf7KPshxnSpxXU2Vec/edit?usp=sharing" target="_blank">
    <b><code> 📄 View Full System Design & Analysis Document → </code></b>
  </a>
</p>

#### -`♡´- Core System Architecture Engineered:
* **Secure Authentication:** Architecture designed to support national identity validation via PhilSys Single Sign-On (SSO).
* **Granular RBAC Matrix:** 4-tier user permission structure isolating data vectors between Citizens, Frontline LGU Staff, Department Heads, and IT System Administrators.
* **Strict Legal Compliance:** Formulated the system specifications around RA 10173 (Data Privacy Act of 2012) and WCAG 2.1 Level AA digital accessibility standards.
